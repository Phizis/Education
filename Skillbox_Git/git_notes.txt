Создание репозитория:

Git-метод: 
создать удаленный реп,
git init - инициализация
git add README.md - добавить файл для коммита
git commit -m "first commit" - непосредственно коммит, если затупил и попал в Vim, Esc-:-wq - *write-quit*
git branch -M main - можно не делать
git remote add origin https://github.com/Phizis/123.git - добавить путь к удаленому репу
git push -u origin main	- запушить изменения

Мой метод:
создать удаленный реп,
git clone - клонировать удаленный в нужную папку, пушить/пулить по ситауции
Разница лишь при работе в команде и больших проектах, где тебе не нужен весь репозиторий
